# 一代战神：“功臣号”

>“功臣”号坦克在 $1950-1953$ 年期间服役于中国人民志愿军，曾在朝鲜战场上痛击“联合国军”，为抗美援朝战争的胜利做出了不可磨灭的贡献。时至今日，“功臣”号早已退役，成为了中国人民的爱国热情、抵御外侮的坚定决心和绝不屈服于帝国主义的铮铮铁骨的象征。

本题将分为三个部分讨论“功臣”号坦克的简化模型。

# $\text{Part 2.}$ 炮台与运动学
